# JavaScript 호이스팅 개념 정리

면접 질문으로 나올 것 같아서 정리

## 호이스팅이란?
변수와 함수 선언이 스코프 최상단으로 끌어올려지는 현상

```javascript
console.log(x); // undefined (에러 안남!)
var x = 5;

// 실제로는 이렇게 동작
var x;
console.log(x); // undefined
x = 5;
```

## let, const는 다름
```javascript
console.log(y); // ReferenceError!
let y = 10;
```

TDZ(Temporal Dead Zone) 때문에 에러남
